## =€ Quick Start

### Installation

```bash
# Clone and build
git clone https://github.com/0xheartcode/git-to-pdf
cd git-to-pdf
cargo build --release

# Or install directly from source
cargo install --path .
```

### Basic Usage

```bash
# Convert a GitHub repository
git-to-pdf https://github.com/user/repo

# Convert local directory
git-to-pdf /path/to/local/repo

# Use dark theme with line numbers
git-to-pdf --theme dark --line-numbers https://github.com/user/repo
```
